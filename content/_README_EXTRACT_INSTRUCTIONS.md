# Extract Instructions

Extract this zip into:

```text
C:\Users\user\dev\quartz\content
```

It adds Markdown wrapper pages for the images already in your Quartz content folder.

After extracting, go back to:

```text
C:\Users\user\dev\quartz
```

Then run:

```cmd
npx quartz build --serve
```

Open:

```text
http://localhost:8080
```

You should see the PACT Player Archive homepage and clickable Prelude handout pages.
